Icons

All icons exist in gif and xpm format. The gif versions are just for documentation (i.e. this document). The xpm versions are used in the application itself. Most icons are used on toolbars but many are also used in menus as well.

Data Architect is cross-platform and this may be a factor in creating icons because there may be different expectations for the icons on the various platforms (UNIX/Linux, Apple, flavours of Windows).

The application uses these icons by including them into the source code at build time. A naming convention exists and must be adhered to.

For example;

AddColumn would be named AddColumn.xpm and would contain a 'C' declaration as "static const char *xpmAddColumn[]". It is likely that the 'C" declaration will have to be edited after saving the image. Also note that all icons have a transparent background denoted by the color "none". The icons sizes are inconsistent at this time but should adhere to standard sizes as soon as possible - starting with a 16x16 set. Image depth can be any reasonable depth that will work on the vast majority of machines - perhaps more than 256 colors.
